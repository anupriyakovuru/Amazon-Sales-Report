# Amazon Sales Analysis Dashboard | Power BI

## Project Overview

This project presents an interactive Power BI dashboard developed using an Amazon Sales Dataset obtained from Kaggle.

The objective of this project is to analyze sales performance, product trends, customer behavior, and revenue insights through data visualization and business intelligence techniques.

---

## Dataset

Source: Kaggle Amazon Sales Dataset

The dataset contains sales transaction information including:

* Order Details
* Product Categories
* Sales Amount
* Quantity Sold
* Customer Information
* Order Status
* Revenue Metrics
* Product Ratings

---

## Business Objectives

The dashboard was designed to answer key business questions such as:

* What is the total sales revenue?
* Which products generate the highest sales?
* Which categories perform best?
* What are the monthly sales trends?
* Which regions contribute the most revenue?
* What factors influence customer purchasing behavior?
